# Prompt Examples

These prompts can be copied directly into Codex. Prompts can be written in English or Spanish — the skill instructions are in English, but you specify the output language.

---

## English Examples

## Illustration Planning Only

```text
Use $ian-xiaohei-illustrations don't generate images yet.
Please analyze the following article and identify where illustrations would be valuable. Output a shot list of about 5 images.
For each image, specify:
- Which paragraph it follows
- Image theme
- Core meaning
- Structure type
- What Xiaohei is doing in the image
- Suggested elements
- Suggested labels (in your target language)

<paste article>
```

## Article Illustrations

```text
Use $ian-xiaohei-illustrations generate 4 whimsical article illustrations for the following article.
Requirements: 16:9 horizontal, pure white background, black hand-drawn line art, sparse red/orange/blue handwritten annotations.
Each image explains only one core structure. No PPT infographics, no cute cartoons.

<paste article>
```

## Long-Form Content Strategy

```text
Use $ian-xiaohei-illustrations create an illustration strategy for this long article.
Don't distribute evenly — only pick cognitive anchors: core judgments, input-output loops, before/after comparisons, common pitfalls, handoff paths.
Default 6-8 images. Output the shot list first, don't generate images yet.

<paste article>
```

## Single Viewpoint Illustration

```text
Use $ian-xiaohei-illustrations generate one 16:9 article illustration for this viewpoint:

Trust isn't shouted — it's paved one piece of evidence at a time.

The scene should be whimsical but clean. Xiaohei must perform the core action.
Handwritten labels: max 5, keep them short.
```

## Workflow Theme

```text
Use $ian-xiaohei-illustrations generate one image for "turning one raw piece of material into three types of content: traffic, trust, and conversion."
Don't draw a formal flowchart. Don't replicate the old multi-use fish example.
Invent a fresh low-tech metaphor. Xiaohei must participate in the core action.
```

## Edit: Remove Title

```text
Use $ian-xiaohei-illustrations help me edit this image.
Remove the "Workflow" title and underline from the top-left corner. Keep everything else exactly the same.
Don't add any new text or objects.
```

## Edit: Enhance Xiaohei's Participation

```text
Use $ian-xiaohei-illustrations this image is on the right track, but Xiaohei looks a bit like decoration.
Keep the core meaning and regenerate a version where Xiaohei is the one actually driving the structure.
Make it a bit weirder, but still pure white, clean, and minimal text.
```

## Generate Style Samples

```text
Use $ian-xiaohei-illustrations output 5 different Xiaohei article illustration style samples.
Topics: information overload, product validation, content compound interest, solopreneur, trust building.
Generate each separately — don't combine into one image.
```

## Spanish Examples / Ejemplos en Español

### Solo planificar ilustraciones

```text
Use $ian-xiaohei-illustrations no generes imágenes aún.
Analiza el siguiente artículo e identifica dónde serían valiosas las ilustraciones. Produce una lista de unas 5 imágenes.
Para cada imagen especifica: después de qué párrafo va, tema, significado central, tipo de estructura, qué hace Xiaohei, etiquetas sugeridas.

<pegar artículo>
```

### Generar ilustraciones de artículo

```text
Use $ian-xiaohei-illustrations genera 4 ilustraciones caprichosas para el siguiente artículo.
Requisitos: 16:9 horizontal, fondo blanco puro, dibujo a mano en negro, anotaciones manuscritas rojas/azules/naranjas escasas.
Etiquetas en español.
Cada imagen explica solo una estructura central. Sin infografías tipo PPT, sin dibujos animados lindos.

<pegar artículo>
```

### Ilustración de un solo concepto

```text
Use $ian-xiaohei-illustrations genera una ilustración para este punto de vista:
"La confianza no se grita — se pisa pieza por pieza de evidencia."
La escena debe ser caprichosa pero limpia. Xiaohei debe realizar la acción central.
Etiquetas en español: máximo 5, que sean cortas.
```

### Editar: quitar título

```text
Use $ian-xiaohei-illustrations ayúdame a editar esta imagen.
Quita el título "Workflow" y su subrayado de la esquina superior izquierda. Todo lo demás permanece igual.
No agregues ningún texto u objeto nuevo.
```
